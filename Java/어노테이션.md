# 어노테이션(Annotation)

###### 2020.01.26

>언노테이션은 실행하고는 상관없이 자바 코드에 주석문처럼 추가하는 부가적인 정보이다.
>
>서로 다른 이름으로 구성된 정보들을 가지는 메타데이터(meta data)

### I. 어노테이션 선언


```java
@interface 어노테이션명 {
    데이터 타입 변수명();
}

// e.g
@interface Check {
    String name();
    int val();
}
```

### II. 어노테이션 사용

```java
@어노테이션명(변수명 = 값, 변수명 = 값)

// e.g
@Check(name="first", val=123)
public static void set() {}
```
