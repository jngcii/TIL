# Maven

###### 2020.06.24

## I. 개요
- 빌드 툴
- 빌드할 때 사용하는 도구
- 빌드 과정을 경량화 시킬 수 있다.

### 프로젝트 빌드 과정
1. 프로젝트 생성
2. 라이브러리 설정
3. 코드 작업
4. 컴파일
5. 테스트
6. 패키지 만들기
7. 배포
8. 레포팅

### 메이븐을 사용하는 이유
- 이클립스에서도 프로젝트를 생성하고 빌드하는 등의 작업을 할 수 있지만, 생산성도 좋고 여러가지 이유가 있다.

## II. 설치
- `https://maven.apache.org`에 접속해 적합한 OS에 맞는 배치파일 다운로드 후 환경변수 설정 (Windows)
- homebrew를 통한 다운로드 (MacOS)
  - 설치하기 : `brew install maven`
  - 버전 확인하기 : `mvn -version`
  - 명령어 위치 확인하기 : `which mvn`
